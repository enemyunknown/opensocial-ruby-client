Class OpenSocial::FetchGroupsRequest
In: lib/opensocial/group.rb
Parent: Request

Provides the ability to request a collection of groups for a given user.

The FetchGroupsRequest wraps a simple request to an OpenSocial endpoint for a collection of groups. As parameters, it accepts a user ID. This request may be used, standalone, by calling send, or bundled into an RPC request.
